 # Suspicious Process Detection System
 A Python-based security monitoring tool that detects suspicious processes.
 ## Features- Real-time process monitoring- Whitelist/Blacklist detection- CPU and memory analysis- Email alerts- Detailed logging
 ## Installation
 1. Clone the repository:
 ```bash
       git clone https://github.com/chamarthinagendravarma-spec/suspicious-process-monitor.git
       cd suspicious-process-monitor
 2. Install dependencies:
     pip install -r requirements.txt
 3. Configure settings:
     Copy  config.json.example to Edit configuration parameters
 4. Run the monitor:
     python process_monitor.py
     config.json
Configuration Edit:
config.json to customize:
 monitoring_mode: "single" or "continuous"
 scan_interval: Seconds between scans
 use_whitelist: Enable whitelist checking
 email_alerts_enabled: Send email notifications
 Usage
 Single scan:
 Set 
monitoring_mode to "single" in config.json
 Continuous monitoring:
 Set 
monitoring_mode to "continuous"
 Stop monitoring:
 Press Ctrl+C
